# 造形規約

template.html の CSS がこの規約の実装形。テンプレートにある部品を使う限り
このファイルを読む必要はない。新しい見た目の要素を足すときだけ、以下の
規約内で作る(v8〜v14 のモック批評で確定)。

## タイポグラフィ

- 本文は約 38 字/行(日本語の適正 35〜40 字。本文 18px なら幅 680px)。
  図・囲み・数式は本文より広い 720px を維持し、本文だけ狭める
- 本文 line-height 1.8
- 見出しは本文サイズ基準の em(h2 = 1.5em、ジャンプ率 150%)
- 見出し前の余白は h2 に付けず、節の頭(padding-top)に寄せる。
  ブロック下余白は 1 種類に統一
- 見出しは `text-wrap: balance` + `word-break: auto-phrase`(文節折り返し)
- 日本語に斜体は使わない(純英語のみ可)
- 章番号は小さな朱ラベル「CHAPTER 00」様式(12px・字間 .25em)。
  役割キッカーも同じ様式
- フォント変数: `--text`(本文)/ `--ui`(ラベル・図解)/ `--mono`

## 強調は 2 系統だけ

- 塗り `em` = 本文の核。下線 = 押せる用語(クリックできるものだけ)
- 太字は列挙の頭とラベルのみ
- 枠線(border)は「操作できる」の専用記号。静的なカードは塗りだけで区切る

## 色の役割(コントラスト)

- `--ink` = 読む文字すべて。補足解説も薄めない(フォントとサイズで階層が
  付くなら色まで薄めるのは二重減衰)
- `--ink-soft` = 一行のラベル・メタ・矢印記号のみ
- `--line-strong`(#978567)= 図の輪郭と操作枠線(非テキスト 3:1 基準)
- `--line` = 区切り罫線のみ。文字と図の輪郭には使わない
- 透過(opacity)で文字をさらに薄めるのは禁止
- 注釈・補足は朱の左罫 3px に 1 本化。色付き囲みは意味の色分け
  (著者の見立て = 朱、既読との接続 = 緑)として別役割で維持
- 角丸は 6 / 10 / 14 / 999 の 4 段のみ

## 目次とナビゲーション

- 左レール・粘着・縦一列・現在地ハイライト。目次は「読む本文の従者」—
  枠なしの静かなリスト(現在地 = 朱の左罫 3px + --ink、ホバー下線)
- レール内の左端の縦線は 2 本まで。子項目のインデントは
  罫 3px + 間隔 12px = 15px だけ
- レール見出しは「目次」(「地図」は心像図に専有させる)
- ページの縦グリッド線は 1 本。表紙にも本文と同じグリッドを適用し
  左端を揃える
- モバイル: リード文直後に非粘着の本文内目次(JS がレールから複製。
  cloneNode はリスナーを写さないのでクリック処理を再付与する)
- スクロールは瞬間ジャンプ + ミニマップ(smooth はプレビュー環境で
  動かないことがある)。粘着ヘッダーに隠れないよう、ヘッダー実高さ + 8px
  を scroll-margin-top に動的設定する

## アニメーション

- ドロップキャップ、章のスクロール出現アニメーションは使わない(v9 で廃止)

## 既知の課題

- モバイルでヘッダーが 3 段(172px)に育つ。ヘッダーに要素を足すときは
  モバイル幅で確認する
